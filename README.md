# update-importmap

Tooling to update importmap.json file from micro-frontends

### Required evironment variables

| Environment variable | Description                                                                    |
| -------------------- | ------------------------------------------------------------------------------ |
| PUBLIC_CDN_URL       | Base url of CDN                                                                |
| CI_PROJECT_NAME      | name of the project which would be used to fetch the JS file from CDN location |
| ORG_NAME             | org name which would be used to update the corresponing section in import map  |
|                      |                                                                                |
